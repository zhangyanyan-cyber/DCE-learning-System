# DCE-learning-System
离散选择实验系统学习资源库是一个面向研究者与学生的一站式 DCE 方法论平台，覆盖从研究设计到计量分析的完整流程。 平台整合了五大学科（医疗卫生、环境经济、交通运输、市场营销、农业食品）的属性与水平参考库，帮助研究者快速确定研究框架；提供正交设计、最大化设计（Federov 算法）与 D-optimal 设计（含贝叶斯版本）三种问卷设计方案的即用型 R 代码；并系统呈现条件 Logit、混合 Logit 与潜在类别模型的完整分析代码，配合 Krinsky-Robb 参数 Bootstrap 法实现支付意愿（WTP）的点估计与置信区间构建。所有代码均附详细中文注释，可直接复制运行。
